Upload files Preparing your test for analysis.

You can drag & drop files or select files from your computer.

Two files are required:

  1. Student test data file contains answers each student gave for each item. Use a CSV format. Each student goes on a separate row and each answer is separated by commas from the next question.
  2. Test version/Answer file contains the correct answers for each item. Use a CSV format. If your test was scanned by the examination office the answer file should be in the format A=01, B=02, C=04, D=08, E=16, with a comma separating each question. Question text is an optional file that contains the text for each item.
